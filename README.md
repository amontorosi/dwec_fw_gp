# dwec_fw_gp

API para pruebas en DWEC y Frameworks en Node.js

## 1. API Houses 

Ampliación del manual 'Build your first Angular app' de la [documentación oficial de Angular](https://angular.dev/tutorials/first-app).

Endpoints disponibles:
- Obtener todas las viviendas: [https://dwec-fw-gp.vercel.app/api/houses](https://dwec-fw-gp.vercel.app/api/houses)

- Obtener vivienda por ID: [https://dwec-fw-gp.vercel.app/api/houses?id=3](https://dwec-fw-gp.vercel.app/api/houses?id=3)
